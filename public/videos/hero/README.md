Leg deine Hero-Videos hier ab.

- Empfohlenes Format: MP4 (H.264) plus WebM (VP9) für bestmögliche Browser-Unterstützung.
- Dateinamen: Passe sie an die im Code hinterlegten Quellen an (`bahnhof-nacht.mp4`, `bahnhof-nacht.webm`, `gleisbau.mp4`, `gleisbau.webm`), oder ändere die Pfade in `app/pages/index.vue` entsprechend.
- Vorschaubild: Wenn du ein eigenes Poster-Bild nutzen möchtest, speichere es ebenfalls hier und trag den Pfad im `poster`-Feld in `app/pages/index.vue` ein.

Alle Dateien in diesem Ordner werden von Nuxt direkt aus `/videos/hero/…` ausgeliefert.
